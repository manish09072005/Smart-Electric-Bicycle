# Smart-Electric-Bicycle
IoT-based Smart Electric Bicycle with GPS Tracking, Battery Monitoring and GSM Alerts.

## Project Overview
This project aims to develop a Smart Electric Bicycle with advanced monitoring and safety features.

## Features
- GPS Tracking
- Battery Monitoring
- GSM Alerts
- Real-Time Location Tracking
- Theft Protection
- Mobile Notifications

## Components Used
- ESP32
- GPS Module
- GSM Module
- Lithium-Ion Battery Pack
- BMS
- DC Motor
- Motor Controller
